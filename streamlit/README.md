# Streamlit Apps

This directory contains Streamlit applications from the Data Professor tutorial series.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run an app:
   ```bash
   streamlit run partX/app.py
   ```
